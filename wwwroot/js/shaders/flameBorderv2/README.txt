Для редактирования шейдера прямо в браузере, необходимо в консоли браузе написать shader_settings.field_name
field_name - название поля, которое нужно изменить.
Все поля shader_settings и сам shader_settings находятся в файле shader_settings.js

Чтобы загрузить настройки, нужно в консоли браузера написать load_settings()
Чтобы сохранить настройки, нужно в консоли браузера написать save_settings()
